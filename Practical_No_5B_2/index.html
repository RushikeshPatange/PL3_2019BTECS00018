Program for Celsius To Fahrenheit conversion

Formula for converting Celsius scale to Fahrenheit scale

T(oF) = T(oC) Ã— 9/5 + 32

Code

C++ code of celcius to fahrenheit conversion

#include <iostream>
using namespace std;
int main()
{
    float fahrenheit, celsius;
    cout<<"Enter temperature in celsius: ";
    cin>>celsius;
    fahrenheit = ((celsius * 9) / 5) + 32;
    cout<<"Temperature in fahrenheit is: "<<fahrenheit;
    return 0;
}

Enter temperature in celsius: 28
